Introduction

YUI3 Undo Framework allows easy adding undo/redo functionality to an application.
It consists of two classes:

  • UndoManager
  • UndoableAction

UndoManager manages a list of UndoableAction-s. In addition to standard undo/redo functionality, YUI3 Undo Framework supports asynchronous actions also. An asynchronous action must fire undoFinished event in order to notify that its undo/redo process has been completed.
