Accordion with overflow

You may also make the whole accordion scrollable, by modifying yui3-accordion class: 

     .acc_container2 {
        height: 500px;
        overflow: auto;
        position: relative;
        width: 450px;
    }

    .acc_container2 .yui3-accordion {
        height: 600px;
        width: 600px;
    }
